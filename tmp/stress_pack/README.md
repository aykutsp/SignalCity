# Human Contextual Stress Index (HCSI)

This package provides a **professor-level prompt framework** and **mathematical modeling base** for quantifying human stress using environmental, economic, and social signals.

## Scope
- Earthquakes & disasters
- News/media intensity
- Traffic density
- Weather & temperature stress
- Economic indicators

## Output
- Composite stress score (0–100)
- Real-time adaptable architecture
- Research-grade methodology

## Structure
- `main_prompt.md` → Master AI prompt
- `formulation.md` → Mathematical model
- `data_schema.json` → Input structure
- `validation.md` → Scientific validation
- `references.md` → APA references
