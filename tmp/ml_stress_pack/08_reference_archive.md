# Reference Archive and Source Notes

## Key source-backed anchors

1. **Perceived Stress Scale (PSS)** is the recommended psychometric anchor for supervised learning labels.
2. **WHO emergency mental-health guidance** supports including disaster and emergency exposure as stress-relevant context.
3. **OECD/JRC composite-indicator guidance** supports structured normalization, weighting, robustness checks, and sensitivity analysis.
4. Traffic, environmental noise, heat, and economic uncertainty each have documented links to mental-health burden or stress-related outcomes.
5. **SHAP** is an accepted explainability framework for deriving transparent contribution patterns from ML models.
